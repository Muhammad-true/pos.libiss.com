# Архитектура Libiss POS (маркетинг)

## Цель
- Одностраничный маркетинговый лендинг для POS.
- Фокус на простоте, скорости и ясном пути: создать магазин → подключить POS.

## Технологии
- Vite (статический сайт).
- Чистые HTML/CSS/JS.

## Структура
- `index.html` — контент слайдов, CTA, футер.
- `src/styles.css` — глобальные стили, scroll-snap, визуальные токены.
- `src/main.js` — переключатель RU/EN, анимации при появлении.
- `src/translations.js` — словарь RU/EN.
- `public/logo.png` — логотип, общий стиль с libiss.com.

## UI-правила
- Каждая секция — полный экран, scroll-snap.
- Минимум текста, крупная типографика.
- Акцентный цвет: мягкий зелёный (как на libiss.com).
- Кнопки большие и тач-дружелюбные.

## i18n
- Переключатель RU/EN хранится в `localStorage`.
- Все тексты размечены `data-i18n` и подставляются из `translations.js`.

## Сборка
- `npm run dev` — локальная разработка.
- `npm run build` — прод-сборка в `dist/`.
- `npm run preview` — предпросмотр `dist/`.
